# Sleep Inhibit Survey
> This is not an exhaustive list (lol), and consists of the games that I currently have installed on my system. This means that Valve (specifically Source 2) games are overrepresented in the data.\
\
I did not test games that are demos, playtests, or otherwise irrelevant

> [!NOTE] 
> Testing methodology: [Diagnostic methods](inhibiting-games.md#diagnostic-methods)

Out of 74 games tested, only 11 actively inhibited system sleep:
- 7 Days To Die
- Aperture Desk Job
- Assetto Corsa
- Counter Strike 2
- Deadlock
- Half-Life Alyx
- Halo Infinite
- Helldivers 2
- MudRunner
- Overwatch
- SteamVR

> [!NOTE]
> Source 2 inhibits sleep by default, making Source 2 games outliers of the dataset.

The 63 games that did not inhibit sleep included games made in Unity, Unreal, Source, as well as numerous proprietary engines/frameworks.
